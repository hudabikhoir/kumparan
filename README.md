# KUMPARAN

This Repo Kumparan Backend candidate test.

## Installation

- clone git repository
``git clone https://github.com/hudabikhoir/kumparan.git``
- setting .env 
- migrate database 
``php artisan migrate``
- Seed database 
``php artisan db:seed``
- test application with php unit run 
``vendor/bin/phpunit``
